# Tezhaveh Gallery - Admin Dashboard

A full-featured admin dashboard for **Tezhaveh Gallery**, a handcrafted pottery and ceramic online store. Built with Next.js, Tailwind CSS, and Supabase.

## Tech Stack

- **Framework:** Next.js (App Router) with JavaScript
- **Styling:** Tailwind CSS v4
- **Database & Auth:** Supabase
- **Forms:** react-hook-form
- **Icons:** lucide-react
- **Charts:** recharts

## Features

- Secure authentication (login/signup)
- Dashboard with sales analytics and charts
- Product management (CRUD with image upload)
- Order management with status updates
- Responsive sidebar and header layout
- Protected admin routes
- Landing page for the store
